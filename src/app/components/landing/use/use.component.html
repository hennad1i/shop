"Shop" - demo application for the purchase of goods.

Application divided by three parts - Landing, User and Admin panels

User and Admin panels protected by default.

To log in as User you need registration (default role - "user") or enter email - "user@gmail.com" and password - "secret" on the authorization form.

User can add to basket and change counts items

To log in as Administrator you need enter email - "admin@gmail.com" and password - "secret" on the authorization form.

Administrator can edit items (only price) and showing fake statistics
